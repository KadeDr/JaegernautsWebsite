Java Setup for Your Robot

Follow these steps to set up Java for your robot:

  1. Download FRC Game Tools from the NI Website
  2. Install WPILib from the FRC Website
  3. Create a Java Project in WPILib
  4. Set up Git Source Control

Step 1: Installing FRC Game Tools

First, visit the NI Website to download the FRC Game Tools. Once there, you will see this page:

Ensure you select the correct game year for the tools you're downloading:
